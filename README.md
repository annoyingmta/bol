# Overview
<b>Bol</b> (bol' - russian боль) is a small toy programming language, where operators consist of only one letter! This allows a creation of really small programs, but at the cost of programmer's sanity.
# Features
- You can name variables using only lowercase letters, this allows up to 26 different variables! Ex:
  - a, b, c, i, j, k, x, y, z...
- You can assign values of variables with strings or other variables, ex:<br>
  ```a"Hello, World!"``` — assign value of variable **'a'** to string **"Hello, World!"** <br>
  ```xy``` — assign variable **'x'** to variable **'y'**, which must have some value.
